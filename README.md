## 🛠️ Technologies

The solutions in this repository are primarily implemented in **C++**. 

I chose C++ as my main competitive programming language due to its unmatched execution speed, granular memory control, and the powerful features of the Standard Template Library (STL). Python is occasionally used as a fallback for specific scenarios involving BigInt operations or heavy string manipulations.
